使用node.js连接SqlSever并进行增删改查操作

HUSH! · 2019-11-16 · 次阅读

前言

最近上完数据库课程，最后的大实验要求连接数据库并对其进行操作。因为对java不是很熟练，所以就选择使用node.js。在之前没有怎么接触过node.js，在做实验的时候一边学习别人的博客，一边熟悉练习。经过几天时间，实现了这个简单的功能。

连接SqlServer

首先需要使用npm 安装SqlServer依赖

然后建一个db.js文件，等会会在其他的文件里面引用这个封装好的函数。

使用node.js写一个接口

这里要注意get和post方法获取前端返回的数据是有差别的。get方法是req.query 而post方法则是req.body 。发送数据到前端则是用的res.json。
最后运行server.js就可以通过接口获取到数据库里面的数据了。

接口地址：http://localhost:3001/自己写的接口名
这里我用Jquery来获取的数据。
